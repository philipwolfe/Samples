.NET Framework SDK Samples

The Microsoft .NET Framework SDK samples demonstrate how to write applications and components that use the Microsoft .NET technologies.

Note: The SDK samples are not supported on Windows 95.  Window 95 is supported at runtime only.  See the SDK release notes for further information.

Installing the Samples
To install the samples, perform the following three steps:
Step 1:  Expand the samples
The .NET Framework samples are located in a self-extracting executable file (samples.exe).  Click on this link to expand the samples to your hard drive. 
 
Step 2:  (Optional)  Install MSDE 
Many of the SDK samples illustrate how to access a SQL Server database using SQL and ADO.NET.  To run these samples, either SQL Server 7.0 (or higher) or MSDE is required.  If you already have one of these installed, skip this step.  If not, click on the above link to install the MSDE 2000 database server.  

Notes: 
·   The MSDE setup program may require a reboot.  After rebooting, return to this page to complete the SDK samples installation.
·   The samples assume that a local SQL Server is using the credentials: username=sa and password=<blank>. The samples will not run properly if your database server uses different credentials.
 
Step 3:  Complete the samples installation
In this final step, the samples setup will import two sample databases into your database server and create several IIS virtual directories.  It will then compile the samples, using NMAKE (which will run in a separate command window).  Please allow NMAKE to finish before continuing.  For more details on the configuration steps the samples setup performs, click here .
 
Note: Depending on your Internet security settings, Internet Explorer may warn you that an Authenticode signature could not be found -- if it does, please select "Yes" to continue.
 
Removing the Samples
Use the Add/Remove Program applet in the Windows Control Panel to uninstall the Microsoft .NET Framework SDK Samples.  Add/Remove Programs can also be used to remove MSDE.
