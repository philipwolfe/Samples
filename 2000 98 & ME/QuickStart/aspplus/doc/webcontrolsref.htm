System.Web.UI.HtmlControls

HTML controls are HTML elements exposed to the server so you can program against them. HTML controls expose an object model that maps very closely to the HTML elements that they render.

System.Web.UI.WebControls

Web controls are server controls with an abstract, strongly-typed object model. Web controls include not only form-type controls such as buttons and text boxes, but also special-purpose controls such as a calendar. Web controls are more abstract than HTML controls, in that their object model does not necessarily reflect HTML syntax.
