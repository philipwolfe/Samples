End Function Public Function <WebMethod(EnableSession:=false)> UpdateAppCounter() As String If Application("HitCounter") = Nothing Application("HitCounter") = 1 Else Application("HitCounter") = CInt(Application("HitCounter")) + 1 End If Return "This service has been accessed " & Application("HitCounter").ToString() & " times."
