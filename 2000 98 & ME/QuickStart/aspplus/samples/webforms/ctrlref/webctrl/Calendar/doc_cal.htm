Date Selection Modes

Calendar supports four date selection modes:

Mode Description
Day User can select any single day.
DayWeek User can select a single day, or an entire week.
DayWeekMonth User can select a single day, or an entire week, or the entire visible month.
None Date selection is disabled.

Adding Custom Content to Calendar

You can make appointment-style calendars by adding content in the OnDayRender event. Two of the arguments for OnDayRender are the Day that is being rendered and its Cell object. Custom text can be added to the cell for a particular day by adding it as a LiteralControl to the Cell's controls collection, as the following example shows. 


string Hol = GetHoliday(Day.Date);

if (Hol != string.Empty) Cells.Controls.Add(new LiteralControl("<br>" + Hol));


Dim Hol As String = GetHoliday(Day.Date)

If Hol <> String.Empty Then Cells.Controls.Add(New LiteralControl("<br>" + Hol))


var Hol:string = GetHoliday(Day.Date);

if (Hol != string.Empty) Cells.Controls.Add(new LiteralControl("<br>" + Hol));
