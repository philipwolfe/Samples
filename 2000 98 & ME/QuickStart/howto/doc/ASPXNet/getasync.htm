How Do I...Make an asynchronous GET request from within ASP.NET?

This sample illustrates using the WebRequest and WebResponse classes to make an asynchronous GET request on a URI from within an ASP.NET application.

This example is exactly the same as the command-line example showing an asynchronous GET request with the net classes. The only difference is that this example shows that you can request another URI from within an .htm page.
