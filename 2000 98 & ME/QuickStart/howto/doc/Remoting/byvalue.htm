How Do I...Pass An Object to a Server By Value?

We mentioned in a previous section that local objects are always passed by value when we call a remote function. Lets change the previous example to demonstrate this.

Lets create the object we need to pass to the server. 


using System;
namespace RemotingSamples {

    [serializable]
    public class ForwardMe {

        private int mValue = 1;

        public void CallMe() {
            mValue++;
        }

        public int getValue() {
            return mValue;
        }
    }
}
C#  

We flag this class with the [serializable] custom attribute to enable it to be streamed to and from the server. When we run the sample, the client reports the value of the counter to be one. Since the server calls the CallMe method on the object 5 times, the value of the counter when the object is returned is 6. It is important to note that the client has 2 instances of the ForwardMe class after calling the remote method, the one that was passed as well as a copy of the one that was returned.
