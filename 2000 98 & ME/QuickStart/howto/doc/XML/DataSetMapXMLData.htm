How Do I...Infer DataSet mappings from XML?

This sample illustrates how to create DataSet mappings from XML Data. Here we use the concept of inference whereby the structure of the XML data is used to build a relational schema. This is then used to create a relational structure of tables and columns to store data in a DataSet that conforms to the inferred schema.


DataSetMapXMLData.htm

[Run Sample] | [View Source]

To achieve inference on the XML the ReadXml method is called with the XML file. 


// Infer the DataSet schema from the XML data and load the XML Data
datadoc.DataSet.ReadXml(new StreamReader("books.xml"));
