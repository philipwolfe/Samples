How Do I...Execute custom code at specified times?

You can create a Schedule component instance for your processing that raises an event on a daily, weekly, monthly, or other calendar-based schedule. You can use this component to automate your business processing. For example, you might create a Schedule component to automatically raise events on the last Friday of every month. Other classes or components in your application might handle the events this component raises, for example, running logic that tabulates tax information for the month, computes monthly sales, and generates a monthly payroll report.

This sample illustrates how to create a Schedule that uses the RecurrencePatternsDialog to define the pattern of when to raise events. Run the sample as follows: 

> Scheduler.exe c:\winnt\notepad.exe

Now, specify the schedule for opening Notepad and click OK. the specified process (Notepad) will start according to the pattern specified in the RecurrencePatternsDialog.

In its simplest form, creating a Schedule that uses the RecurrencePatternsDialog involves:

  1. Creating a new instance of Schedule: 

    
Schedule aSchedule = new Schedule();

    
Dim aSchedule As Schedule = New Schedule()
    C# VB  

  2. Creating a new instance of RecurrencePatternsDialog: 

    
RecurrencePatternsDialog dlg = new RecurrencePatternsDialog();

    
Dim dlg As RecurrencePatternsDialog = New RecurrencePatternsDialog()
    C# VB   

    
dlg.RecurrencePatterns = aSchedule.RecurrencePatterns;
dlg.ShowDialog();

    
dlg.RecurrencePatterns = aSchedule.RecurrencePatterns
dlg.ShowDialog
    C# VB  

  3. Setting the patterns specified through the dialog into the Schedule: 

    
aSchedule.RecurrencePatterns.All = dlg.RecurrencePatterns.All;

    
aSchedule.RecurrencePatterns.All = dlg.RecurrencePatterns.All
    C# VB  

  4. Specifying the event to raise: 

    
aSchedule.EventOccurred += new OccurredEventHandler(OnScheduledEvent);

    
AddHandler aSchedule.EventOccurred, AddressOf OnScheduledEvent
    C# VB  

  5. Enabling the component: 

    
aSchedule.Enabled = true;

    
aSchedule.Enabled = true
    C# VB  

  6. Defining the event to raise: 

    
public static void OnScheduledEvent(Object source, OccurredEventArgs e) {
    Process.Start(executablePath);		
}

    
Public Sub OnScheduledEvent(ByVal source As Object, ByVal e As OccurredEventArgs)
    Process.Start(executablePath)
End Sub
    C# VB
