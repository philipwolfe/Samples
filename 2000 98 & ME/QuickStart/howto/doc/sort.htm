How Do I...Sort an array?

This sample illustrates sorting the elements in an integer array, a double array, and a string array. Using the Array.Sort method, you can sort any array whose elements implement IComparable. You can also sort arrays of your own types as long as they implement IComparable. If a type does not implement IComparable (or if you want to customize the sort order) you can still use Array.Sort if you provide a Comparer, such as the GuidComparer shown in the sample.
