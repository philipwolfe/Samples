How Do I...Throw an exception?

This sample illustrates how to throw an exception. A try..catch..finally block is used in this sample. Any code that may throw an exception is placed inside of the try block. If an exception is thrown then the catch block is entered and the program can perform some operation to recover or alert the user. The code in the finally block is always executed and can be used to perform cleanup after an exception has occurred.
