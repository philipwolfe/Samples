Using the Windows Forms Designer

The .Net Framework SDK includes an application called WinDes. The WinDes application provides form designer, Properties window, and Toolbox to perform design time testing of Windows Forms controls that you have authored. You can use these tools to add your control to a form and set its properties in the property browser. In addition to providing a way to view and set your control's properties, the Properties window also allows you to view custom designers and UI Type Editors you have defined for your control.

In addition to providing a way to test the controls you have designed, you can also use the WinDes Form Designer to create the user interface for a Windows Forms application. WinDes uses the same underlying code base as the Windows Forms designer in Visual Studio.

Running WinDes

To run WinDes, navigate to the bin directory where you have installed the .Net Framework SDK. Type WinDes.

Creating Forms

In Windes, you can add a Form by clicking New C# or New VB from the File menu. Once the form designer is displayed, you can add controls to the form by double-clicking on a control in the Toolbox or by selecting a control in the Toolbox and drawing the control on the form. You can set the properties of a control by selecting the control in the form designer and setting the properties in the Properties window. To create an event-handling method for an event of a control double-click on the control. This will create an even-handling method for the default event of the control. You can also create an event-handling method by using the Events tab (Lightning Bolt) on the Properties window.

Adding Controls to the Toolbox

You can add your controls to the Toolbox for use with the WinDes Form Designer.

  1. Click Add Library from the Edit menu. 
  2. In the dialog box displayed, select the DLL that contains the control you want to add to the Toolbox.

The control will appear at the bottom of the Toolbox.
