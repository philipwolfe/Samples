Advanced .NET Framework (GDI+): Use Animation

This application demonstrates how to do animation with GDI+, including classic frame animation, drawing and moving a shape on the screen, and animating text with a gradient fill.

Featured Highlights

  • Example 1: Classic frame animation using four images which are cycled back and forth to show a winking eye.
  • Example 2: A bouncing ball animation in which the size and speed of the ball are relative to the client area.
  • Example 3: Animated text using color animation effected by moving the gradient fill start and end points.

Requirements

Microsoft Visual Studio.NET Professional
Windows 2000 or Windows XP.

Running the Sample

Simply press F5.
