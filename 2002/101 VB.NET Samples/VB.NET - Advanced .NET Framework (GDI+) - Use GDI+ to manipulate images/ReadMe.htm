Advanced .NET Framework (GDI+): Manipulating Images with GDI+

This sample application shows how to manipulate images using GDI+.

Featured Highlights

  • Browse for and open images in a PictureBox
  • Set the PictureBox SizeMode
  • Zoom In and Out, and Fit the image in the PictureBox according to the image's true aspect ratio
  • Rotate images left and right 90 degrees
  • Crop images
  • Save the PictureBox image to a file, either at normal size or as a thumbnail using the GDI+ GetThumbnailImage method

Requirements

There are no special requirements to run this sample.

Running the Sample

Simply Press F5.
