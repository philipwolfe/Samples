.NET Framework: Working with GDI+ -- Pens

This sample shows the majority of the features available when using the Pen object when using GDI+.

Featured Highlights:

  • The user can select from a number of different parameters that influence how the pen draws.
  • After each selection, a group of lines are redrawn to demonstrate the new pen.
  • Some more advanced features that are demonstrated are :
    - Basing the Pen object on a Brush object, so that advanced lines can be drawn.
    - Animating a dashed pen.
    - Skewing a pen's width so that it behaves like a calligraphy pen.

Requirements:

Microsoft Visual Studio.NET Professional or greater.
Windows 2000 or Windows XP

Running the Sample:

Simply Press F5.
