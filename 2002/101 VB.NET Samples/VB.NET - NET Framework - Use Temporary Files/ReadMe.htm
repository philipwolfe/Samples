.NET Framework: Creating Temporary Resources

This sample shows some how to create and use temporary files, as well as locate the Temp directory on the user's machine. The creation and use of these resources in the .NET Framework is very simple. 

Featured Highlights:

This tutorial demonstrates how to create and use temporary files. In particular, it discusses:

  • Finding the Temp directory on a user's machine.
  • Creating a temporary file using the Path.GetTempFileName() method.
  • Setting a temporary file attributes to 'Temporary' to improve performance.
  • Delete a temporary file.

Requirements:

Microsoft Visual Studio .NET Professional or greater.
Windows 2000 or Windows XP.

Running the Sample:

Simply press F5.
