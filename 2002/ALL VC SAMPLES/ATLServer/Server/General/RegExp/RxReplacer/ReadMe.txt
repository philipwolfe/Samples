Instructions:
1.	Build and deploy the RxReplacer Solution(you can deploy by right clicking on the deployment project(rxReplacerDeploy) and selecting deploy. This will create a new virtual directory on your local machine.
2.	Use a browser to navigate to http://localhost/rxreplacer/ and view the output.
3.  Choose the file in which you want to find/replace your regular expression using the browse button, type in the regular expression and the new expression you want to substitute it with.
4.  Click on "Get Modified File" to obtain the new file.
