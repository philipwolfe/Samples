COMEXCEL is a stand-alone Automation client. 
It creates a new Microsoft Excel spreadsheet and 
generates a pie chart of the data stored on that spreadsheet. 
This sample works only with Microsoft Excel, which exposes the dispinterfaces.
To build this sample within Visual C++, determine which version of Excel you 
have on your machine and then look in the appropriate COMEXCEL sample directory.
Make sure that the #import reference to COMEXCEL.exe is in the sample source code is correct for your computer.
If you get an access violation while running the COMEXCEL sample under 
Windows NT 4.0, please install the Windows NT Service Pack 3.
