Microsoft BizTalk Server 2000 enables the development and management of application integration within and between organizations using the universal standard of XML. BizTalk Server 2000 provides comprehensive process management and is a platform for reliable business document interchange and business processes integration.
 

Microsoft BizTalk Server 2000 Enterprise Beta

Microsoft® BizTalk™ Server 2000 Enterprise Edition enables the development and management of business processes within and between organizations. Companies can easily build distributed business processes that integrate applications and partners, as well as quickly establish reliable, secure trading relationships with customers and key partners over the Internet. By supporting important Web technologies such as XML and SOAP, BizTalk Server 2000 allows an organization to leverage open standards and specifications in order to integrate disparate applications independent of operating system, programming model or programming language.

The BizTalk Server 2000 Beta features the following enhancements from the Technology Preview:

  • NEW! BizTalk Orchestration visual business process modeling environment, based on Microsoft Visio 2000.
  • NEW! BizTalk Orchestration business process execution engine, based on XLANG.
  • Easier to use management and configuration tools, including many new UI Wizards.
  • Support for BizTalk Framework v2.0 Draft Specification for reliable messaging.
  • Additional capabilities in BizTalk Mapper to enable more complex data transformations
Microsoft® BizTalk Server™ 2000 Enterprise Edition Beta System Requirements
  • An Intel Pentium 300 Processor
  • 128 MB of RAM
  • A 6GB hard disk
  • Microsoft® Windows® 2000 Server or Windows® 2000 Advanced Server
  • Microsoft® SQL Server™ 7.0 Standard (Service Pack 2 or later) or Microsoft® SQL Server™ 2000 Standard Edition Beta 2
  • Microsoft Visio® 2000 Standard Edition SR1a is required to run BizTalk Server Application Designer
  • Microsoft® Office 2000 Web Components are required for Document Tracking UI.

If you do not meet the requirements above, order the BizTalk Server 2000 Beta Kit. This Kit contains the software that you need to evaluate BizTalk Server.

Microsoft® Commerce Server™ 2000 Hotfix
IMPORTANT! In order to enable seamless purchase order exchange between the Commerce Server 2000 Beta and the BizTalk Server 2000 Beta, you must install the Commerce Server 2000 hotfix. BizTalk Server 2000 Beta does not yet support Internet Explorer 5.5, whereas Commerce Server 2000 requires this browser version for the Business Desk functionality. Therefore, in setting up these beta products, you are encouraged to configure them on separate physical computers. This issue will be resolved before both products launch their final versions. In order to download the Commerce Server hotfix to ensure proper PO exchange, please visit: http://download.microsoft.com/download/comserver/Install/2000/NT45/EN-US/Q269469_cs2k_bts.exe.

If you do not already have a copy of the Commerce Server 2000 Beta, you may download it from: http://www.microsoft.com/commerceserver/prodinfo/beta.asp.
