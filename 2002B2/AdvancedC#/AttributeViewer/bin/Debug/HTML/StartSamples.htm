NGWS SDK Samples

The Microsoft NGWS SDK samples demonstrate how to write applications and components that use the Microsoft NGWS technologies.

Explore the Samples

The NGWS samples are located in a self-extracting executable file (samples.exe) located under the Samples directory. You can install the samples by clicking the link below and responding to a few questions. The samples will be installed in subdirectories under the Samples directory.

Install NGWS SDK Samples

Note: Until you run samples.exe to extract the sample files, the links to the individual samples listed below will not function properly.

Note: Depending on your Internet security settings you may be prompted about running this program by Internet Explorer.
Note: Depending on the password security settings defined on your machine you may see a dialog during the install process alerting you that SQL Server has created a password that does not meet the minimum password security settings on your machine. Press okay on this dialog and setup will continue.

Uninstalling the Samples

How to uninstall the samples
  • Run the sample configuration utility with the uninstall switch

      For Example:
      C:\Program Files\NGWSSDK\Samples\Setup\:>configuresamples -u

  • Delete the Samples directory.
  • To remove the Microsoft Database Engine.
    Use the Add\Remove Control Panel option to remove MSDE from your machine.



Manually Building and Configuring the Samples

Manually configuring the samples requires the creation of a number of IIS Virtual Roots, the importing of several databases into SQL Server and the addition of a few registry keys.  The NGWS SDK samples require a machine running IIS 5.0 and SQL Server 7.0.  The samples assume that a local SQL Server is using the credentials: username=sa and password=<blank>.  The samples will not run properly if your database server uses different credentials.  If SQL Server is not installed on your machine, run the default installation and an MSDE database server will be installed for you.
 
Installation Instructions
  • Run samples.exe to extract the NGWS SDK samples directory structure.  UnCheck the "When Done Unzipping Run Setup\ConfigureSamples" option. 
  • Run the NMAKE program to build the samples from the command line:
      • For example:
      C:\Program Files\NGWSSDK\Samples>NMAKE ALL
  • Complete the Manual Configuration Instructions to create the IIS VRoots, configure the databases and add the registry keys.
