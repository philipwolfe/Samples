Web DataSet Demo

The Web DataSet Demo requires that SQL Server 7.0 or SQL Server 2000 be installed on the 
local machine and contain the Northwind DB.

To install and run the WebDataSetDemo:

1.  Copy the WebDataSetDemoSetup.msi file from the WebDataSetDemo folder to your computer.

2.  Run the WebDataSetDemoSetup.msi file and follow the directions.

3.  Open the .sln file

	Note:  If VS.NET can't open the project via Front Page Server Extensions, a warning dialog will 
	       be displayed asking to switch to file based access.  Choose OK to continue opening the project

4.  Press F5 to run the sample

	Note: If you are prompted to set the Start Page for the application, right click on the 
	      WebForm1.aspx item in the Solution Explorer and select Set As Start Page.
