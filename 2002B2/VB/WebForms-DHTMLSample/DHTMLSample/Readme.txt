Web Form DHTM Sample

To install and run the WebFormDHTML Sample:

1.  Copy the WebFormDHTMLSetup.msi file from the WebFormDHTML folder to your computer.

2.  Run the WebFormDHTMLSetup.msi file and follow the directions.

3.  Open the .sln file

	Note:  If VS.NET can't open the project via Front Page Server Extensions, a warning dialog will 
	       be displayed asking to switch to file based access.  Choose OK to continue opening the project

4.  Press F5 to run the sample

	Note: If you are prompted to set the Start Page for the application, right click on the 
	      WebFormDHTMLDemo.aspx item in the Solution Explorer and select Set As Start Page.
