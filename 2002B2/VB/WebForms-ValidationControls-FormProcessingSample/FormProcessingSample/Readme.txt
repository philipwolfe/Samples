Visual Basic.NET - Form Processing

To install and run the Form Processing sample:

1.  Copy the FormProcessingSetup.msi from the WebForms-ValidationControls-FormProcessingSample folder to your computer.

2.  Run the FormProcessingSetup.msi file and follow the directions.

3.  Open the .sln file

	Note:  If VS.NET can't open the project via Front Page Server Extensions, a warning dialog will 
	       be displayed asking to switch to file based access.  Choose OK to continue opening the project

4.  Press F5 to run the sample

	Note: If you are prompted to set the Start Page for the application, right click on the 
	      Register.aspx item in the Solution Explorer and select Set As Start Page.

5.  Click the submit button before entering any data to see the validation rules

6.  Enter the information to register and click submit
