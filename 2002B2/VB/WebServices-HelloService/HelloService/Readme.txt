Visual Basic.NET - Hello World Web Service Sample

To install and run the Hello World Web Service sample:

1.  Copy the HelloServiceSetup.msi from the WebServices-HelloService folder to your computer.

2.  Run the HelloServiceSetup.msi file and follow the directions.

3.  Open the .sln file

	Note:  If VS.NET can't open the project via Front Page Server Extensions, a warning dialog will 
	       be displayed asking to switch to file based access.  Choose OK to continue opening the project

4.  Press F5 to run the sample

5.  Click the Say Hello button
