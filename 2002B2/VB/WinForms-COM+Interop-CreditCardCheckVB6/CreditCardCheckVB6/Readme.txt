Visual Basic.NET - Credit Card Check VB6



To install and run the Credit Card Check VB6 sample:

1.  Register the VB6 Dll located at \Credit Dll\CreditCard.dll using regsvr32.exe

2.  Open the .sln

3.  Press F5

4.  Enter a credit card number (you can get a valid number by clicking the test number button)

5.  Click the verify number button to see if its valid.
