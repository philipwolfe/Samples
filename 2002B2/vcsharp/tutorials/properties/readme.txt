This sample shows how properties are an integral part of the C# programming language. It demonstrates how properties are declared and used.

This sample contains the source code for the Properties Tutorial.

Building and Running the Sample Within Visual Studio

To build and run the Properties samples


	Open the solution (Properties.sln).

	In Solution Explorer, right-click the person project and click Set as StartUp Project.

	From the Debug menu, click Start Without Debugging.

	Repeat steps 2 and 3 for shapetest.


Building and Running the Sample from the Command Line

To build and run the Properties samples


	Use the cd command to change to the person directory.

	Type the following:
csc person.cs
person


	Use the cd command to change to the shapetest directory.

	Type the following:
csc abstractshape.cs shapes.cs shapetest.cs
shapetest
