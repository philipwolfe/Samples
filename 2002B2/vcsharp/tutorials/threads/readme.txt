This sample demonstrates various thread activities: creating and executing a thread, synchronization of threads, interaction between threads, and using a thread pool.

This sample contains the source code for the Threading Tutorial.

Building and Running the Sample Within Visual Studio

To build and run the Threading samples


	Open the solution (Threads.sln).

	In Solution Explorer, right-click the StopJoin project and click Set as StartUp Project.

	From the Debug menu, click Start Without Debugging.

	Repeat steps 2 and 3 for MonitorSample and SimplePool.


Building and Running the Sample from the Command Line

To build and run the Threading samples


	Use the cd command to change to the Threads directory.

	Type the following:
cd StopJoin
csc StopJoin.cs
StopJoin


	Type the following:
cd ..\MonitorSample
csc MonitorSample.cs
MonitorSample


	Type the following:
cd ..\SimplePool
csc SimplePool.cs
SimplePool
