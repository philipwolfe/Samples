This sample demonstrates versioning in C# through the use of the override and new keywords. Versioning maintains compatibility between base and derived classes as they evolve.

This sample contains the source code for the Versioning Tutorial.

Building and Running the Sample Within Visual Studio

To build and run the Versioning sample


	Open the solution (Versioning.sln).

	From the Debug menu, click Start Without Debugging.


Building and Running the Sample from the Command Line

To build and run the Versioning sample


	Type the following at the command prompt:
csc versioning.cs
versioning
