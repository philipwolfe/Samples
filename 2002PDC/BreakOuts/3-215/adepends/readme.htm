Adepends is a tool that shows the structure of an assembly and its dependencies.  By default, the information is shown in graphical form, but the -C option can be used to send the output to the console.  The command line to adepends includes a file containing the manifest for the assembly you'd like to display.  For example: "adepends math.exe". 

This tool is unsupported in the PDC Tech Preview.  It will likely be included in the SDK in a future release.
