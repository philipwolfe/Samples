ADO Interoperability Sample

 

This document contains the release information for the ADO Interoperability Sample. 

 

Sample Overview

This sample demonstrates the use of an existing unmanaged COM component (ADO) from ASP Plus page using C# and VB7. It interacts with the ADO COM server and uses it’s objects and methods.

 

Requirements:

This sample assumes IIS services and SQL server are configured and running on your system.

This samples requires msado15.dll to be registered.

 

Location of the samples:

This sample is contained in two subdirectories:

Sample1 – Using ADO from an ASP Plus page using C#

Sample2 – Using ADO from ASP Plus page using VB7

 

Building and Running the Sample

 

Run the build.bat. From each subdirectory run runme.bat

Internet Explorer starts the ASP Plus page. It displays the initial records in the SQL table ‘titles’, updates a record, and new record is added to the titles table. Before you close Internet Explorer the records are changed back to their initial status and the added record is deleted.
