Managed Events Interoperability Sample

This document contains the release information for the Managed Events Interoperability Sample.  You will find the following sections below:

  • Sample Overview
  • Location of the Sample
  • Building and Running the Sample

 

Sample Overview

This sample demonstrates sinking managed events in VB6.   

Location of the Sample

This sample is contained in Sample\ManagedEvents directory. The managed server is in Sample\ManagedEvents \EventSrc directory. The VB6 client is Sample\ManagedEvents\VBSink directory.

Building and Running the Sample

To build this sample, run “build.bat” batch file in the project directory. The result of this batch file are eventsrc.dll and EventSrc.tlb. The EventSrc.dll is the managed dll. EventSrc.tlb is its type library. Open the VB6 project in VB6 IDE, add EventSrc.tlb to the project. Then build the VB project and put the result in the same directory as EventSrc.dll.

To run the sample, run the exe produced by VB compiler after building the VB project. There will be a window with three buttons. Clicking the Start button will start the clock. Clicking the Stop will stop the clock.
