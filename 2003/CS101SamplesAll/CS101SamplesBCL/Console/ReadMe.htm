Console Sample

The new members of the Console class in the .Net framework 2.0 adds a number of new features. You can now change the window and buffer size in code as well as manipulate the change the foreground and background colors and add a title. You can now also move rectangular areas of the screen buffer which can be used in simple animations, control the input by checking for specific key presses and change the size and visibility of the cursor. Finally you can set the frequency and duration of the console beep.

In this sample we use all of the new features in an interactive walk-through. As you follow the instructions you will see the new functionality in action.

This sample requires the following to be installed:
  • Visual Studio 2005.
