Creating Setup CAB

The .NET Compact Framework (.NET CF) 2.0 and Visual Studio 2005 make it easier to deploy smart device applications.

This sample demonstrates how to create a CAB file for an application. 

Deployment is the process by which you transfer an application to the target device.  Before you can deploy your solution, you must package it into a CAB file. A CAB file is a type of executable archive file that contains the application, dependencies such as DLLs, resources, help files, and any other files you want to include.

This SetupCAB project creates and inserts registry values and an application shortcut.  The CABDemo application demonstrates reading the registry values inserted by the CAB file.  Microsoft Visual Studio 2005 provides tools for packaging your CAB file. It does not provide any tools for deploying the CAB file to a target device or to the emulator.  Drag the CAB file from your desktop machine to your device using an ActiveSync connection.  After the CAB file is copied to the target device, tap on it to explode the CAB file and begin the installation process.

This sample requires the following to be installed:
  • Visual Studio 2005 Beta 2.
  • .NET Compact Framework v2.0 (installed on physical device)
