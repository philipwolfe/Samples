Registry Sample

The .NET Compact Framework (.NET CF) 2.0 and Visual Studio 2005 make it easier to create smart device applications.

This sample demonstrates how to use the registry on the Pocket PC.

Before .NET CF 2.0, the registry could only be accessed with p/invoke.  In this sample, we demonstrate how to create registry keys and subkeys and add names and values to these keys.  In order to protect other registry settings on your device, this sample only allows registry manipulation under the HKEY_CURRENT_USER\SOFTWARE\RegistrySample\ tree.

This sample requires the following to be installed:
  • Visual Studio 2005 Beta 2.
