Context Enabled Form

One of the most powerful uses of Microsoft Windows XP Tablet PC Edition 2005 is for form-related activities. In most cases, this can be done without coding. The "Context Tagging Tool" provided with the Tablet PC Platform SDK allows you to specify field "masks" for use with any form-based application, without altering the source code. The Microsoft Windows XP Tablet PC Edition 2005 searches for a specially-formatted XML file when loading an executable; this XML file can contain context-sensitive masks or hints for the build-in Tablet Input Panel.

This sample illustrates the context-sensitive capabilities of Microsoft Windows XP Tablet PC Edition 2005 explicitly in code. It shows values of the Factoid type and allows them to be selected. The code explicitly disallows the "Word List" factoid, which cannot be programmatically used with the PenInputPanel type (although it can be used using the Context Tagging Tool). When a factoid value is selected, the PenInputPanel is used in conjunction with a standard TextBox control to demonstrate the context hinting.

This sample requires the following to be installed:

  • Visual Studio 2005.
  • Microsoft Tablet PC Platform SDK.
  • Microsoft Windows XP Tablet PC Edition 2005.
