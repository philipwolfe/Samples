Power Awareness

One of the keys to a highly mobile smart client is awareness of the power status of the machine. Certain graphics, file and network operations can be battery-intensive and, if possible, a smart client application should be able to alter its behavior to accomodate different power situations.

While power-awareness is not part of the .NET 2.0 Base Class Library, it is fairly straightforward to read via P/Invoke. This sample provides some power-aware classes and enumerations and demonstrates their use; reading the current charging status and battery strength.

This sample is not specific to the Microsoft Windows XP Tablet PC Edition 2005 and does not require the Tablet PC Platform SDK.

This sample requires the following to be installed:

  • Visual Studio 2005.
  • Microsoft Windows XP.
