Creating Smart Tags

Smart Tags allow you to create handlers in your documents for recognizing types of words, and offering relevant actions based on those words.  Microsoft Office includes Smart Tag handlers for elements such as postal addresses and names.

In this sample, you see a Smart Tag that can recognize product ID from the AdventureWorks database.  This is accomplished using a regular expression that can recognize the pattern of XX-XXXX or XX-XXXX-##.  If the pattern is detected in a document, the ID is underlined, and Smart Tag actions for displaying or inserting the relevant product information are available in the context menu.  The information is retrieved from the AdventureWorks database using a typed dataset.  The custom Smart Tag also takes advantage of a feature that allows it to find specific words in addition to the pattern, in this case, "TEST."  Finally, the Recognize method is overridden to demonstrate where more advanced recognition could be used.  In this example, the recognition is simply delegated back to the base class.

As an example, you can try LN-1024 or FR-R92B-58, or any other product ID's from the Product table.

This sample requires the following to be installed:

  • Visual Studio 2005
  • Microsoft Word 2003
  • Adventure Works sample database installed at "C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data"
