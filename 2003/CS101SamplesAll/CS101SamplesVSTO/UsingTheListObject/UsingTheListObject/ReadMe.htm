Using the ListObject in Excel

The ListObject is a key object in Excel.  They offer a container for ranges that support a header row, automatic total calculation, and dynamic growth as rows are added.  Built in filtering and sorting rounds out the feature set.  Learning about the ListObject provides an important tool in modeling data in a logical and intuitive manner for your users.

In this sample, the main sheet is setup as a simple invoice for the ficticious company, Adventure Works.  The list of ordered products is implemented as a ListObject.  In order to add products, a user control, AddProductUserControl is hosted in the Document Actions pane.  Enter a product number from the Product table, lookup the number, then you can insert the number, name, and price into the ListObject.  This uses a strong-typed dataset for the database integration.  The logic is all contained in the user control, other than the creation and presentation of the control in the Startup event of the sheet.  You can try out the sample using a product number of SA-M198.

This sample requires the following to be installed:

  • Visual Studio 2005
  • Microsoft Excel 2003
  • Adventure Works sample database installed at "C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data"
