Working with XML in Word

Visual Studio Tools for Office 2005 makes it easy to add an ActionsPane control to your Word documents. These controls can contain numerous Windows Forms controls to provide a rich user interface experience.

In this sample, you will see how to create and add an ActionsPane control that allows you to work with XML from an RSS feed. The ActionsPane control has a TextBox for entering the URL to an RSS feed. When the Go! button is clicked, the XML is retrieved, processed into a DataTable, and then displayed in the Word document using Bookmark controls.

This sample requires the following to be installed:

  • Visual Studio 2005
  • Microsoft Word 2003
