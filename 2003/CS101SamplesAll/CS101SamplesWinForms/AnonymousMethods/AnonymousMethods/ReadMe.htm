.NET Framework: Anonymous Methods

This sample uses shows how Anonymous Methods can be used to simplify the code required when using delegates.

Featured Highlights

This Windows Form application provides a default list of 'grocery items' that can be added to as desired and a set of organizing tools that can be used to manipulate the list data.  When looking at the code note that the use of delegates has been simplified dramatically when performing operations on the List<T> generic type.  In addition to removing the need for a separate method declaration they are also declared in line with the calling method of the List<T> class.

Note the following when running the application:

  • Add Item—Enter an item description and a price then click the 'Add' button to display the item in the grid and append it to the List<T>.  When an item is added the List<T>.ForEach method is used with an Anonymous Method to repopulate and revise the total price. 
  • Filter—Select a filter method from the pull down box and enter a price.  When the 'Filter' button is clicked the List<T>.FindAll method compares the given criteria with the items in the list and returns a new List<T> with only the matching items.
  • Sort---Select an item to sort by with the pull down box and click the 'Sort' button.  Items will be sorted alphabetically according the selected criteria. 
  • Reset---The grid can be reset after performing a filter or sort by clicking the 'Reset' button.

Requirements

Requires Visual Studio 2005.

Running the Sample

Press F5 to start the application. The list of default items will be loaded automatically.
