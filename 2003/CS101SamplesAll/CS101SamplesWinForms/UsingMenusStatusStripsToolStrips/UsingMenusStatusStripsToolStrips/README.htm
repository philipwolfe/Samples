Using ToolStrips, MenuStrips, and StatusStrips

The ToolStrip, MenuStrip, and StatusStrip controls are new in .NET Framework 2.0 They replace the ToolBar, MenuBar, and StatusBar controls of the .NET Framework 1.x.

The sample application is a simple text editor. It contains a MenuStrip control, two TabStrip controls, a ContextMenu control for the RichTextBox, and a ProgressBar control.

This sample requires the following to be installed:

  • Visual Studio 2005
