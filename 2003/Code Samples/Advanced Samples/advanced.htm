Advanced Samples

Multithreading - How-To Async Calls
This solution demonstrates how to use multiple threads to achieve concurrency in tasks. This example does not include synchronization, because no data is being accessed by multiple threads.

Remoting - How-To TCP Remoting
This sample is designed to show how to use the .NET Remoting architecture using three types of remote objects. In addition, it uses XML configuration files to specify available objects and server location information instead of hard-coding them into the binaries or putting the data into the system registry.

Advanced: Serializing & Deserializing Objects
This solution demonstrates how to serialize and deserialize an object to/from a stream.

Interacting Windows Service
This sample shows some how to control an installed Windows Service from code.

Make WIn32 API Calls
How to use Win32 API functions.

Animation with GDI+
This application demonstrates how to do animation with GDI+, including classic frame animation, drawing and moving a shape on the screen, and animating text with a gradient fill.

Create a Screensaver with GDI+
This sample shows some how to create a screen saver using Visual Basic.NET.

Use GDI+ to manipulate images
This sample application shows how to manipulate images using GDI+.

Working with GDI+ Brushes
This sample shows many of the features available when using the Brush object in GDI+.

Working with GDI+ Text
This sample shows some of the many features available when using GDI+ to work with text.

Work with Resource Files
This sample shows how to work with resource files and the "localizable" property of the form to localize an application.

(Networking) - Use Sockets
This demonstration shows you how to use Sockets in C#.

(Threading) -- Thread Pooling
Shows how to use Thread Pooling in C#.

(Windows Services) - Create a Windows Service
This sample shows some how a Windows Service is created and deployed.
