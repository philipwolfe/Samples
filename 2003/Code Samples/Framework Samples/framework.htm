More Framework Samples

How-To Configuration Settings
This sample demonstrates how to read and write to an application's configuration file appSettings section.

How-To Environment Settings
This solution demonstrates the use of several classes that provide environment information, from the System.Environment namespace.

How-To MSMQ
This sample shows how to send a MSMQ message from a Windows Forms client that contains a object instance serialized using XML. The messages are received and processed by a Windows Service.

How-To Process Viewer
This sample uses various classes in System.Diagnostics to gather running process information.

How-To Reflection
Demonstrates how to reflect against assemblies to discover type information.

How-To Service Manager
This sample shows how to control Windows Services under Windows NT, Windows 2000, or Windows XP.

How-To Stack Frame
This sample demonstrates two uses of the StackTrace and StackFrame classes: it shows how you can walk all the way back up the stack (all the way back up, as you'll see) at any point, and it shows how to walk the stack after an exception.

How-To System Events
Investigate the SystemEvents class, which allows your applications to react to several events raised by system activity.

How-To Work with XML
This sample shows how to take advantage of the XMLDocument class built-in to System.Xml.dll. This class exposes the XML Document Object Model, or DOM, which can be used to manipulate XML data.

Key Benefits
To demonstrate some of the benefits of VB.NET.

Partitioning your application
Demonstrates How-To partition an application. This enables the developer to be much more productive since the details are encapsulated in another component and only need to be updated in 1 place, instead of multiple apps.

Role Based Security with Enterprise Services
Show how to add Role-based security to components running within the Enterprise Services environment. Such security controls who has access to the component, based on roles defined within the COM+ application to which the component belongs, and having users added to those roles.

Framework - Scoping, Overloading, Overriding
Shows how to allow various levels of access to the members of a class, including Public, Private, Protected and otherwise. To demonstrate how to extend derived classes with features like Overloading and Overriding.

Understanding the Garbage Collector
This sample shows some how to control how the Garbage Collector operates. It demonstrates when the garbage collector calls any Finalize methods on created objects, as well as how to supress finalization.

Using the COM Port
This how-to demonstrates how to control a communications port from .NET. Since the .NET Framework does not support direct control of the Comm Port, a separate class is used that provides direct Win32 API calls. (Note: Comm Port is synonymous with COM Port, Serial Port or RS232 port.)

Using WMI
How to use Windows Management Instrumentation (WMI).
