Interop: Automate Internet Explorer

Demonstrates how to automate Internet Explorer. 

Featured Highlights:

Shows how to use Internet Explorer as an ActiveX Control
Shows how to automate Internet Explorer and set various properties such as size, viewable toolbars, etc.

Requirements:

Microsoft Visual Studio.NET Professional or greater.
Windows XP Home or Professional

Running the Sample:

Simply press F5.
