Create and use Trace Listeners
How to use the built in trace listeners to trace and debug code. Also, how to build and implement a custom trace listener.

How-To Send Mail
This sample shows how to send email over SMTP using classes in the System.Web.Mail namespace.

How-To Use the EventLog
The purpose of this demonstration is to show you how to interact with the Event Log.

How-To Working with GDI+ Pens
This sample shows the majority of the features available when using the Pen object when using GDI+.

Read and Write Performance Counters
This sample shows some how to read from and write to Performance Counters using Visual Basic.NET. These counters can be used to track the performance of various parts of the operating system and .NET applications.

Reading and Writing with a Text File
This sample shows some how to use the StreamReader and StreamWriter classes to read and write text files.

Use Temporary Files
This sample shows some how to create and use temporary files, as well as locate the Temp directory on the user's machine. The creation and use of these resources in the .NET Framework is very simple.

Use the Process Class and Shell Functionality
Illustrates some of the ways in which you can launch and monitor processes using the Process class found in the System.Diagnostics namespace.

Work with Console Applications
This sample shows how to create console applications.
