Step 3 of 7: Calendars

The Calendar control can fire off code each time a new date is selected. This is useful for loading data or changing the properties of another control.

  1. Paste the following code into the code-behind file 
    Private Sub OrderDateStart_SelectionChanged(_
               ByVal sender As System.Object, _
               ByVal e As System.EventArgs) _
               Handles OrderDateStart.SelectionChanged
                  
                  txtStartDate.Text = _
                  OrderDateStart.SelectedDate.ToShortDateString
            
End Sub

Private Sub OrderDateEnd_SelectionChanged(_
               ByVal sender As System.Object, _
               ByVal e As System.EventArgs) _
               Handles OrderDateEnd.SelectionChanged
               
                  txtEndDate.Text = _
                  OrderDateEnd.SelectedDate.ToShortDateString
                  
End Sub
  2. This code simply demonstrates how straightforward it is to get the currently selected date from the calendar and use it in your code. Here we are simply setting the Text property of the TextBoxes, but you could also use it to update or retrieve records from a database.
