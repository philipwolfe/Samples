Step 5 of 7: Format Date

The DataGrid exposes events that allow you to format the data before it is displayed. Using the ItemDataBound event, we can format the OrderDate field using a standard format, such as ShortDate.

Paste this function into your code-behind.

Private Sub dgOrders_ItemDataBound(ByVal sender As Object, _
      ByVal e As System.Web.UI.WebControls.DataGridItemEventArgs) _
      Handles dgOrders.ItemDataBound

        Dim itemType As ListItemType
        Dim intCell As TableCell

        itemType = CType(e.Item.ItemType, ListItemType)

        If ((itemType = ListItemType.Item) Or _
                (itemType = ListItemType.AlternatingItem)) Then
            intCell = e.Item.Cells(3)
            intCell.Text = Format(intCell.Text, "Short Date")
            intCell = e.Item.Cells(4)
            intCell.Text = Format(intCell.Text, "Short Date")
            intCell = e.Item.Cells(5)
            intCell.Text = Format(intCell.Text, "Short Date")
        End If

    End Sub
