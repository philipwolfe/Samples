Step 5 of 5: Dataset Properties

Our Datagrid will use data from the Orders table, but the purpose of our application is to find only a subset of that data. ADO.NET introduces a useful concept called a DataView. A DataView is exactly what it sounds like, a particular view on your data. Think of it as a filter based on certain criteria you choose. For instance, all customers from a certain state or all orders from a certain date range, like the application we are building. We will explore how to create a dataview when we add the code.

  1. We are going to name our dataview dvOrders and it will be the source of data for our DataGrid.

  2. Click the dgOrders DataGrid to view properties.

  3. We will be setting 3 properties.

  4. Set the DataSource property to dvOrders

  5. Set the DataMember property to Orders
  6. Set the DataKeyField property to OrderID
