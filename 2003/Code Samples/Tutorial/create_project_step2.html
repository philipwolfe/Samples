Managing Solutions, Projects, and Files

Step 2 of 3: Create Project

  1. To get things started, open Visual Studio .NET 2003 and create a new project.

  2. From the File menu, choose New - Project. A New Project dialog box will be displayed. The left half of the dialog displays project types and the right half displays project templates which serve as the starting point for new applications.

  3. Under Project Types, choose Visual Basic Projects.

  4. Under Templates, choose ASP.NET Web Application.

  5. In the Location field, type the name http://localhost/OrdersWebVB and select OK.

  6. In the Solution Explorer, you will see your new project. A Web Form named WebForm1.aspx will be created for you.
