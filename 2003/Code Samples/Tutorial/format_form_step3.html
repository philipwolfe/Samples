Step 3 of 4: PropertyBuilder

A more advanced wizard called the Property Builder can assist you in setting various properties for your datagrid. Some of the properties you can set include:

  • Specific style options for your datagrid, as well as for specific sections such as headers, footers and row items.
  • Borders, gridlines and spacing of cells
  • Datasource, sorting and paging options

We will use the PropertyBuilder to add a EditCommandColumn. An EditCommandColumn will use the first column of the datagrid to provide an Edit link to put the row into Edit mode. When in Edit mode, the column will contain Update and Cancel links.

  1. To access the PropertyBuilder, right-click on the datagrid and choose PropertyBuilder... from the menu.

  2. On the left side, click Columns to set various column properties for the datagrid.

  3. Notice that Create columns automatically at run time is checked.

  4. Under Column List, in the Available Columns area, expand ButtonColumn.

  5. Highlight Edit, Update, Cancel and click the > button to move it to the Selected Columns

  1. In the Column Properties, set the HeaderText property to Edit.

  2. Click OK to close the PropertyBuilder.
