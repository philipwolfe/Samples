Compare Validator

An important aspect of creating Web pages for user input is to be able to check that the information users enter is valid. Within the .NET Framework are a set of validation controls that provide an easy-to-use but powerful way to check for errors and, if necessary, display messages to the user. A Compare Validator control is used to compare a user's entry against a constant value, or against a property value of another control, using a comparison operator (less than, equal, greater than, and so on).
