ErrorProvider component

The ErrorProvider component allows you to show the user in a non-intrusive way that something is wrong. An error provider is a better alternative than displaying an error message in a message box, because once a message box is dismissed the error message is no longer visible. The ErrorProvider component displays an error icon next to the relevant control, such as a text box; when the user positions the mouse pointer over the error icon, a ToolTip appears, showing the error message string.
