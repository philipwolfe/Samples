DriveInfo Sample

The DriveInfo class in the .Net framework 2.0 provides an easy way to access the properties of all the drives on a system and return size, name, type, directory and volume information.

In this sample we use all of the properties of the DriveInfo class to simulate the properties window available in Explorer when selecting a drive and choosing properties frrm the option menu. The DriveInfo class provides an instance method that returns a DriveInfo object for each drive on the computer.

This sample requires the following to be installed:
  • Visual Studio 2005.
