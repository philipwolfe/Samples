FTP Sample

The System.Net namespace adds full support for the FTP protocol, making it easy to create full featured FTP clients in .NET.

In this example we use the FtpWebRequest and FtpWebResponse to provide the ability to download files from an FTP site. To connect to a secure FTP site enter the Username and password and then click Set. Make sure to uncheck the Use Anonymous User option to send the credentials to the server.

This sample requires the following to be installed:
  • Visual Studio 2005.
