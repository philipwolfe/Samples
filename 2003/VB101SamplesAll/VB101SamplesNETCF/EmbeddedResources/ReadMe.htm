Embedded Resources

The .NET Compact Framework (.NET CF) 2.0 and Visual Studio 2005 make it easier to create smart device applications.

Resources such as files and images may be embedded into the assembly and accessed at run time from the assembly.  A couple of notes to ensure you embed your desired resources correctly. 

First, add the desired resource to the project--in our case a .bmp image file.

Second, set the Build Action property of the resource to 'Embedded Resource.' 

Third, when  referencing the resource in code, be sure and remember to include the namespace in the resource name when calling the GetManifestResourceStream method.  The namespace for this project is 'EmbeddedResources' which is the name of the project.  Furthermore, if resources are placed in a subdirectory under the project root directory such as 'Images,' then the correct call to the GetManifestResourceStream method would be something like this: GetManifestResourceStream("EmbeddedResources.Images.Gone Fishing.bmp")

This sample uses an embedded resource image to set the background of the form and embedded resource images to load the thumbnail PictureBox controls.  Clicking on a thumbnail image will display this image in the MainPictureBox.

This sample requires the following to be installed:
  • Visual Studio 2005 Beta 2.
