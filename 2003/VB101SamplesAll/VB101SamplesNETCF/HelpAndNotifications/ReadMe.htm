Help and Notification

The .NET Compact Framework (.NET CF) 2.0 introduces a number of controls for building smart devices applications that deliver more features and are easier for consumers to use.

This sample demonstrates ways to use the Start -> Help event to display help for a form as well as different uses for the Notification object.

Click the Start Button to see a notification popup based on a timer that processes an HTML response.

Use the context menu of the track bar to see tool tip style context sensitive help using a notification popup.

Use the Pocket PC Start Menu -> Help to see a notification popup.

This sample requires the following to be installed:
  • Visual Studio 2005 Beta 2.
