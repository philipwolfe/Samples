Using MSMQ

The .NET Compact Framework (.NET CF) 2.0 and Visual Studio 2005 make it easier to deploy smart device applications.

This sample demonstrates the new support for the Microsoft Message Queue (MSMQ) in the .NET Compact Framework (.NET CF) 2.0.

In order to use MSMQ on a device or with the VS 2005 emulator, the following files from the SDK must be present in the windows subdirectory: msmqadm.exe, msmqadmext.dll, msmqd.dll and msmqrt.dll.  These four files are added to the project with the "Build Action" property set to "Content" and the "Copy To Output" property set to "Copy Always."  The first time this sample is run, the files are copied to the windows directory and the MSMQ is initialized using the msmqadm utility through a CreateProcess call to the CoreDll.dll.

This sample demonstrates saving and retrieving strings to the queue.  Simply add string messages to the queue by typing text in the TextBox and clicking the "Send" button.  Click the "Receive" button to get the available messages from the queue and display them in the ListBox.

This sample requires the following to be installed:
  • Visual Studio 2005 Beta 2.
