Controlling Drawing Attributes

The Microsoft Tablet PC Platform SDK provides complete control over the visual style of drawn ink. Attributes include the obvious, like color and height and width of the "pen tip", and less obvious, such as transparency, raster operations, pressure sensitivity, and "pen tip" shape. This sample demonstrates the most common way of manipulate these values, by setting the DefaultDrawingAttributes property of an InkOverlay object.

This sample requires the following to be installed:

  • Visual Studio 2005.
  • Microsoft Tablet PC Platform SDK.
  • Microsoft Windows XP Tablet PC Edition 2005.
