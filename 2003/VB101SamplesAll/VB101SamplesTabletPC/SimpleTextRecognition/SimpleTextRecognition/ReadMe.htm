Simple Text Recognition

The Microsoft Tablet PC Platform SDK provides text recognition capabilities in the Microsoft.Ink namespace. Text recognition is performed by RecognitionContext which are produced by an instance of type Recognizer. Multiple Recognizers may (and probably will) be installed on a single machine. Often, you will want to use the default recognizer, which can be retrieved with the GetDefaultRecognizer() method of the Recognizers class.

This sample enumerates the Recognizers installed on the current system and allows the user to choose between them. An InkOverlay is attached to a Panel for ink collection. After inking some text, a "Recognize" menu item triggers text recognition. The best guess or "top string" is placed in a ListBox and then all the alternates are placed in the ListBox as well.

The user can additionally erase the ink.

This sample requires the following to be installed:

  • Visual Studio 2005.
  • Microsoft Tablet PC Platform SDK.
  • Microsoft Windows XP Tablet PC Edition 2005.
