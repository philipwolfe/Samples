Working with Contacts and Tasks in Outlook

Outlook is a powerful tool for managing email, contacts, tasks, and more.  Each item type offers a powerful programming model for interaction making it easy to integrate custom logic with pre-built functionality.  Contacts support a wealth of person-related information, while tasks allow you to define actions that need to be taken.  Both support a number of meta fields including due date, subject, reminders, and more.

In this sample, you will create a menu command to enter messages to call people back.  A ComboBox will present the list of contacts from the default Contacts folder, along with a button to create a new contact.  The large, multi-line TextBox provides an area to enter a message, and a MaskedTextBox allows a callback date to be entered.  All of these fields are presented in a custom Windows form.  When the Save button is clicked, a new Task item is created with a reference to the specified contact, along with the optional message body and due date.  Discover some of the available Outlook events and the proper methods of creating and interacting with Outlook items.

This sample requires the following to be installed:

  • Visual Studio 2005
  • Microsoft Outlook 2003
