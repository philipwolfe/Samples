Using the DataGridView

This sample demonstrates the use of the DataGridView control, a powerful new grid control for displaying data in a table format. It replaces the DataGrid control.

The DataViewGrid control is bound to the Employee data from the Adventure Works database. Notice the variety of cell controls. For example, try clicking on the Subordinates button for Peter Krebs, and then selecting one of the employees from the pop-up ListBox.

This sample requires the following to be installed:
  • Visual Studio 2005
  • SQL Server 2005 Express Edition
  • The AdventureWorks database installed in the default location
