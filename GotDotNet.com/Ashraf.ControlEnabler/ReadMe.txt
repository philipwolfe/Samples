** ASP.NET Page Control Enabler Class **

Often it becomes useful to disable or enable certain type of controls. For instance, we may need to make disable or read-only all the text boxes in a page when the current user has permission as read-only. The utility class included here will help you in that context, along with having opportunity to make enable or disable all the controls of an ASP.NET page.
