Vasesoft FileUpload Web Control Specification


Platform:
Windows 2000, Windows XP
.NET Framework with ASP.NET ver 1.0

Installation:
Copy FileUpLo.dll to the web application /bin directory.

Usage:
1. Register the FileUpload web control to the current .aspx page
<%@ Register TagPrefix="VaseWeb" Namespace="Vasesoft.WebControls" Assembly="FileUpLo"%>

2. Define the web form for posting “multipart/form-data” data
<form id="FileUp" method="post" encType="multipart/form-data" runat="server">

3. Place the FileUpload web control to the page, and set the control’s property.
<VaseWeb:FileUpload id="ClientFile" SavePath="e:\upfile" RequiredField="True" MaxFileSize="100K" ValidFileExtensionName="jpg;jpeg;gif" SaveMode="feedback" runat="server" />

SavePath : the directory the upload file save to
RequireField : Set to true, the file upload field is blank will cause error message.
MaxFileSize : The limit of the file maximum size. For example, "100K", "1M", "1024B" or "512"
ValidFileExtensionName : The acceptable file extension name, separated by ";", not include "." . For example, "doc;txt;xls"
SaveMode : The method to handle saving the same name file name at server, Feedback,AutoRename,OverWrite is available. Default value is AutoRename.

  
4. Program for more function
　

private void Button1_Click(object sender, System.EventArgs e)

{

     if (ClientFile.IsValid)

     {

         if (ClientFile.SavedFileName != "")

         {

              Label1.Text = "File is uploaded." + "<br>";

              Label1.Text += "Save filename as: " + ClientFile.SavedFileName  + "<br>";

              Label1.Text += "Save to path: " + ClientFile.SavePath + "<br>";

              Label1.Text += "Save file fullname: " + ClientFile.SavedFileFullName + "<br>";

              Label1.Text += "File size: " + ClientFile.FileSize + "<br>";

              Label1.Text += "Save date: " + ClientFile.SavedFileDate + "<br>";

              Label1.Text += "Posted file name: " + ClientFile.PostedFileName + "<br>";

              Label1.Text += "Posted file path: " + ClientFile.PostedFilePath + "<br>";

         }

     }

     else

         Label1.Text = ClientFile.ErrorMessag;

}
