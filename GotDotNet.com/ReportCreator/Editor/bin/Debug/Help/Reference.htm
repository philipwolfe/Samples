Arrow keys

Move to the next cell and add it to selections

Band title click

Select all cells of the band

Ctrl + Band title click

Add all cells of the band to selections

Select bands

Select any cell

 

Shift + Band title click

Select all bands between last selected and current

Resize cells and bands

[Ctrl + Arrow keys] or
[Mouse move while cursor is V/H split]

Change width/height of current cell/band step by Grid size X/Y
(see Options | Preferences))

Shift + (see above)

Change width/height of current cell/band step by 1 pixel

Alt + (see above)

Adjust border between two cells/bands
