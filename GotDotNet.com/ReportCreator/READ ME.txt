Installation
--------------------------------------------------------------------------------
Open ReportEngine.sln and built it.
For Excel Export i am using Microsoft Office 2003. If you want use earlier 
version of Office  you must reference it.

Demo
--------------------------------------------------------------------------------
Nice samples for ReportCreator. 

Editor
--------------------------------------------------------------------------------
A report template editor. You can edit a template file (*.sd) using this 
application.

Viewer
--------------------------------------------------------------------------------
A report viewer tool. You can open a report file (*.sdr) using this 
application.

VSIP Package
--------------------------------------------------------------------------------
You can edit a report template file in Visual Studio ide. Read the Vsip
package file for installation notes.
