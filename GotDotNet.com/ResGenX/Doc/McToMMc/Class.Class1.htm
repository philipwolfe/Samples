SummaryConvert .mc file for use with managed code
 
The syntax of the McToMMc command is: 
McToMMc infile.mc[,outfile.ext] ...
The infile must have an extension of mc . If the output file is not specified a name of infile.mmc will be used. Multiple infile,outfile parameters may be specified.

The %c formating strings used in an mc file are replaced as follows:

%.
Replace with a period.
%b
Replaced with "\b".
%%
Replaced by a percent sign.
%!
Replaced by a exclamation point.
%r
Replaced with "\r".
%\
Replaced with "\n".
%0
Replaced with "\0".
%nn[!printf-format-specifier!]
Replaced with a managed code format specifier.
