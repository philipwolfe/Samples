SummaryContains the routines that do the actual processing.
 
The routines in ResGenx are:
Process
Evaluates the command line
GenResFromMC
Loads the ResGenMC class which performs the conversion of an .mc file to a resource.
GenResFromTxt
Converts a .txt file to a resource
GenTxtFromRes
Converts a resource into a .txt file
GenResFromRes
Convert a .resources to a .resx or visa versa
