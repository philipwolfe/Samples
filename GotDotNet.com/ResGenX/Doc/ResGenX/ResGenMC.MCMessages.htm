ResGenX.ResGenMC.MCMessages

CodeType Function
Access Private
 
SummaryProcess the messages in the .mc file and write them to a resource file
SyntaxSystem.Int32 MCMessages()
 
A message definition consists of the following 
 MessageId = [number|+number]
 Severity = severity_name
 Facility = facility_name
 SymbolicName = name
 OutputBase = {number}

 following by: (for each language)
 Language=name
 message text
 .
The OutputBase keyword is ignored. Severity and Facility are optional and default to Success and Application respectivly.
