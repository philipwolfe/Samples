ResGenX.ResGenX.GenTxtFromRes

CodeType Function
Access Private
 
SummaryGenerate a txt file from a resources file.
Syntax
void GenTxtFromRes(System.String inFileName,
                   System.String outFileName,
                   System.String resType
                   )
 
inFileName
The name of the input file.
outFileName
The name of the output file.
resType
The type of resource file to create
 
Only string elements of the resource file are written to the output file. A multi line text message will be generated under the following condiftions:
  1. The text length must be greater that 100.
  2. If the message text contains the newline sequence \n the text will be split at that the point immediately following the newline sequence.
  3. If the text does not contain newlinr characters, it will be broken into segments of 80 characters maximum. If possible the break point will ne on a space. The max line length, 100, and the max segment length, 80, are configurable thru the application configuration file.
