ResGenX.Util.Message

CodeType Function
Access Project
IsShared
 
SummaryWrites messages to the console
Syntax
void Message(System.Resources.ResourceManager rmgr,
             System.String msgID,
             System.Object[] args
             )
 
rmgr
The ResourceManager for the message resources
msgID
The ID of the message to write
args
Any string format arguments
 
This routine looks up a message in the Messages resource and writes it to the console. Error messages are written to stderr all other messages are written to stdout .
