XPath XPress Help

Summary

XPath XPress is an Xpath expression evaluation tool written in the Microsoft.NET framework.

Getting Started

  1. Load an XML document you wish to evaluate expressions against by clicking on the 'open' button in the toolbar. A treeview representation of the document is displayed on the form.
  2. Set the context node for evaluating the expression against by clicking on one of the nodes in the tree. If you do not select a node the root node of the document will be used. The context node is highlighted when you evaluate an expression.
  3. Type your expression in the text box at the top of the form. XPath XPress can evaluate the expression as you type it, or only when you click the 'Test' button. You can set this behaviour in the preferences dialog which is displayed when you click the preferences button in the toolbar.
  4. Matching nodes are hilighted in the document tree. The number of matches is displayed in the status bar at the bottom of the form. Also, you can move to any of the matching nodes at any time by selecting them from the drop-down list in the toolbar.

Preferences

The following properties can be set in the preferences dialog.
  • Context Node Color: The color that is used to hilight the context node in the document tree view.
  • Match Node Color: The color that is used to hilight nodes that match your expression.
  • Evaluate Expressions Interactively: Set the way expressions are evaluated, either as you type them, or when you click the test button.

XPath Xpress uses the the DotNetWidgets controls developed by windows forms code hero Tim Dawson. They are available from his site http://www.divil.co.uk/net/
