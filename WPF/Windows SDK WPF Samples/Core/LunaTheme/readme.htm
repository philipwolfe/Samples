Luna Theme

[This readme is a partial copy of the sample's topic in the SDK documentation.]

These are the XAML files that define the Luna theme for Microsoft .NET Framework version 3.0 controls on Windows Vista.

Remarks

The Luna theme is used for Windows XP style and includes three color variants: Blue (NormalColor), Olive Green (Homestead), and Silver(Metallic).
