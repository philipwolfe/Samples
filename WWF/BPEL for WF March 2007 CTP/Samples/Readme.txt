HOW TO RUN THE SAMPLES
----------------------

To execute the samples follow this steps:

- Open in the solution file in Visual Studio.
- Press F5.
- If the sample is based in a console project, it will execute and close when finished.
- If the sample is a Web project wait until the Service Web Page prompt appears, type any string and invoke.
- The Output.txt file should appear in the solution directory with the expected results.
